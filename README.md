# Minion Mayhem 🍌

A Brotato-inspired top-down survival game starring minions. Survive **5 minutes** of escalating waves, auto-banana enemies, and level up with roguelike upgrades.

## Play

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080

## Controls

| Input | Action |
|-------|--------|
| WASD / Arrow keys | Move |
| — | Auto-attacks nearest enemy |

## Gameplay

- **5-minute timer** — survive until time runs out to win
- **Waves** — enemies spawn in waves that get harder over time
- **XP orbs** — dropped by kills; level up to pick one of three upgrades
- **Bosses** — appear every 5 waves and in the final minute

## Upgrades

Damage, attack speed, move speed, HP, multishot, pierce, XP magnet, regen, splash damage, crit chance, lifesteal, and armor.
